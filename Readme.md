# array-without

  Return a copy of an array and remove all passed values

## Installation

  Install with [component(1)](http://component.io):

    $ component install bredele/array-without

## Usage

```js
var without = require('without');
without(['olivier', 'bruno', 'amy'], 'bruno', 'amy');
// ['olivier']
```
